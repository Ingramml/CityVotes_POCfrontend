Sample Data Format

Your JSON file should contain a votes array with the following structure:

{
  "votes": [
    {
      "agenda_item_number": "7.1",
      "agenda_item_title": "Budget Amendment",
      "outcome": "PASS",
      "tally": {
        "ayes": 5,
        "noes": 2,
        "abstain": 0,
        "absent": 0
      },
      "member_votes": {
        "Mayor Smith": "AYE",
        "Council Member Jones": "NAY"
      },
      "meeting_date": "2024-01-15",
      "meeting_type": "Regular"
    }
  ]
}
Valid Values:
  • outcome: PASS, FAIL, FLAG, CONTINUED, REMOVED, TIE
  • member_votes: AYE, NAY, ABSTAIN, ABSENT, RECUSAL
